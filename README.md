# Bài Tập Thực Hành 1 (BTTH1)

Repository này chứa mã nguồn C++ cho các bài tập thực hành cơ bản.

## Thông tin sinh viên

- Bảo Trình
- **MSSV:** 25521929

## Danh sách bài tập

- `bai1.cpp`: Viết chương trình nhập vào một phân số, rút gọn phân số và xuất kết quả.
- `bai2.cpp`: Viết chương trình nhập vào hai phân số, tìm phân số lớn nhất và xuất kết quả.
- `bai3.cpp`: Viết chương trình nhập vào hai phân số. Tính tổng, hiệu, tích, thương giữa chúng và xuất kết quả.
- `bai4.cpp`: Chương trình nhập vào một ngày, kiểm tra tính hợp lệ và tìm ngày kế tiếp.
- `bai5.cpp`: Chương trình nhập họ tên, điểm Toán, Văn và tính điểm trung bình.
